About

These workshop lessons and info pages are meant to be supplemental and catered towards extra professional development. We hope to make this a space for everyone to learn and contribute to!

How can I contribute?

We encourage contributions of all kinds, even pull requests for clarifying language and typo fixes. The goal is to get you comfortable with contributing to a git repository.

💡 If you're not familiar with git, try this resource: [url coming soon]

If you're comfortable with git already, it's recommended to check out the README.md of this repository. Try adapting this lesson style first! However, you are also welcome to create your own style if you would like.
